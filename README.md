# GeoSupply AI Simulator

A multi-page Streamlit web app for geopolitical supply chain risk simulation, energy-AI nexus analysis, and semiconductor stock impact forecasting.

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Pages

1. **Home** — Overview, methodology, quick-start scenarios
2. **Geopolitical Simulator** — Configure shocks, visualize global trade disruptions
3. **Energy-AI Nexus** — Explore datacenter energy constraints and carbon intensity
4. **Stock Impact** — Analyze public and private company exposure to supply shocks
5. **Data Explorer** — Browse raw datasets with quality filters

## Data Sources

Global AI Sovereignty & Hardware Supply Chain dataset (Kaggle).
